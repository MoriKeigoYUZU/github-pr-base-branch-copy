# Privacy Policy — GitHub PR Base Branch Copy

**Last updated:** April 13, 2026

## Data Collection

This extension does **not** collect, store, or transmit any user data.

## Permissions

This extension requires access to `github.com` solely to add a copy button to Pull Request pages. It does not access any other websites.

## Third-Party Services

This extension does not use any third-party services, analytics, or tracking.
